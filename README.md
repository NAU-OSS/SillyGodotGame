This Project is a small Godot game (SillyGodotGame) -- A platforme, right now, but in the future, maybe more?

IF you'd like to work on learning opensource and game design in an new enviroment consider joining!

Installation - Coding
1) We are using Godot .NET, 4.6-stable, you can find it on Godot's website [HERE](https://godotengine.org/download/archive/)
2) Clone this repo
3) Open the Godot Editor -- Click import in the top left -- click on the folder where you cloned this repo

Installation - Playing
1) See the Releases tab of this repo
